Detailed Description

A Real Time Socket CAN device.

Most harware in Linux use the /dev interface. Typically, your device (usb, serial port, ...) will be represented by a file in /dev. CAN hardware is no exception except for the socket CAN module. SocketCAN implements a CAN stack and extend the BSD sockets just like you have a ethernet stack and sockets. RT SocketCAN is the "real time" version for the Xenomai framework. To use osaRTSocketCAN, you must have a Xenomai patched Linux kernel with the RT CAN module enabled
