Detailed Description

template<typename _class, typename _elementType>
class cmnConditionalObjectFactoryOneArg< CMN_DYNAMIC_CREATION_SETNAME, _class, mtsGenericObjectProxy< _elementType > >

Specialization of cmnConditionalObjectFactoryOneArg with enabled dynamic creation. Requires default constructor and SetName method. This emulates a constructor with an std::string parameter and is provided for backward compatibility. Here _elementType should be std::string.
