Detailed Description

template<typename _class, typename _argType>
class cmnConditionalObjectFactoryOneArg< CMN_DYNAMIC_CREATION_ONEARG, _class, _argType >

Specialization of cmnConditionalObjectFactoryOneArg with enabled dynamic creation. Here, _argType must be derived from cmnGenericObject. Note that the cisstMultiTask library removes this requirement by using automatic wrapping with mtsGenericObjectProxy.
