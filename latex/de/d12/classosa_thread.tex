\subsection{Detailed Description}
Define a thread object. 

The thread class is just a wrapper on the O\+S-\/specific thread management functions. For now, wraps functions such as Create\+Thread, Delete\+Thread, Get\+Thread\+Priority, Set\+Thread\+Priority and so on. Additional functionality, such as inter-\/thread communication, maintenance of state information, etc., are provided by the \hyperlink{classmts_task}{mts\+Task} class.

Delay/suspend execution of the calling thread for the specified time. On some platforms, this method provides a better implementation of osa\+Sleep, provided that it is called from the thread associated with this instance of the class. In this case, the Sleep will also be terminated by any call to Wakeup. If called from a different thread, it just calls osa\+Sleep and is therefore not affected by any calls to Wakeup.
