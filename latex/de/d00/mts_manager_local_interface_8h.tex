\subsection{Detailed Description}
Definition of \hyperlink{classmts_manager_local_interface}{mts\-Manager\-Local\-Interface}

This class defines an interface used by the global component manager to communicate with local component managers. The interface is defined as a pure abstract class to support two different configurations that this interface is used for\-: Standalone mode\-: Inter-\/thread communication, no I\-C\-E. A local component manager directly connects to the global component manager that runs in the same process. In this case, the global component manager keeps an instance of type \hyperlink{classmts_manager_local}{mts\-Manager\-Local}.

Network mode\-: Inter-\/process communication, I\-C\-E enabled. Local component managers connect to the global component manager via a network proxy. In this case, the global component manager keeps an instance of type mts\-Manager\-Proxy\-Server.
