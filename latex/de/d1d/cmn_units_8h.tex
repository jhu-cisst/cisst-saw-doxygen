\subsection{Detailed Description}
Declaration of units and unit conversion methods

This file include the definition and implementation of constants and global functions used to defines measuring units in the cisst libraries. The internal units of the cisst libraries are meters, grams, seconds and radians. 

For all units, the cisst libraries provide a constant as well as a global function to convert from the internal units to other representations.

{\ttfamily  double distance = 5000 $\ast$ cmn\+\_\+m; // 5000 meters, distance is actually in mm An\+Example\+Of\+Function(50 $\ast$ cmn\+\_\+mm); // 50 millimeters std\+::cout $<$$<$ \char`\"{}distance in cm\+: \char`\"{} $<$$<$ cmn\+Internal\+To\+\_\+cm(distance) $<$$<$ std\+::endl; }

\begin{DoxyNote}{Note}
The use of underscore in symbol names is an exception to the cisst naming conventions. 
\end{DoxyNote}
