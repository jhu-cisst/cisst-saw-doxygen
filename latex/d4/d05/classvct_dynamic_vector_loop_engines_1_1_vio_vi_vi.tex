Implement operation of the form $v_{io} = op_{io}(v_{io}, op_{v}(v_{i1}, v_{i2}))$ for dynamic vectors. 

This class uses template specialization to perform store-\/back vector-\/vector-\/vector operations

\[ v_{io} = \mathrm{op_{io}}(V_{io}, \mathrm{op_{vv}}(v_{i1}, v_{i2})) \]

where $v_{io}$ is an input-\/output (store-\/back) vector; $v_{i1}$ and $v_{i2}$ are two input vectors. A typical example is\-: $v_{io} += v_{i1} \cdot v_{i2}$. $op_{vv}$ is an operation between the elements of $v_{i1}$ and $v_{i2}$; $op_{io}$ is an operation between the output of $op_{vv}$ and the elements of $v_{io}$.
