\subsection{Detailed Description}
S\-A\-W component for establishing a network connection via Open\-I\-G\-T\-Link protocol.

The component may be configured as a server by providing a target hostname/\-I\-P and a port number (delimited by \char`\"{}\-:\char`\"{}), or as a client by simply omitting the host.

\hyperlink{classmts_open_i_g_t_link}{mtsOpenIGTLink} server(\textcolor{stringliteral}{"trackerServer"}, 50.0 * \hyperlink{cmn_units_8h_aaf4d3f2fafb9b4a95606544b9d876b4a}{cmn\_ms});
server.Configure(\textcolor{stringliteral}{"18944"});
\hyperlink{classmts_open_i_g_t_link}{mtsOpenIGTLink} client(\textcolor{stringliteral}{"trackerClient"}, 50.0 * \hyperlink{cmn_units_8h_aaf4d3f2fafb9b4a95606544b9d876b4a}{cmn\_ms});
client.Configure(\textcolor{stringliteral}{"localhost:18944"});
\end{DoxyCode}
