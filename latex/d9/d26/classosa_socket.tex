\begin{DoxyReturn}{Returns}
Number of bytes received. 0 if timeout is reached and/or no data is received. 
\end{DoxyReturn}
\begin{DoxyNote}{Note}
It can be difficult to determine when a complete set of packets has been received. This function is intended primarily to receive the packet stream produced by Send\-As\-Packets, so it terminates reception in the following cases\-: (1) when a received packet is smaller than the specified packet\-Size, (2) on any socket error, and (3) when a timeout occurs. The first condition is best, since it avoids the possibility of appending data from an unrelated packet. Thus, the caller can ensure that the total number of bytes transmitted is not a multiple of the packet\-Size (e.\-g., by appending an extra byte).
