\subsection{Detailed Description}
Define a Mutex object. 

Mutex class provided to create mutual exclusion around critical sections. This class relies on Posix threads mutex when possible. On Windows, it uses an actual Mutex which provides a locking mechanism between threads as well as processes. It doesn\textquotesingle{}t rely on the more specialized Critical\+Section type (between threads only).
