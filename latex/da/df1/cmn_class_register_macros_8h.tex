\subsection{Detailed Description}
Class registration macros. 

These macros can be divided in three sets, one of each should be used for any registered class.


\begin{DoxyItemize}
\item \hyperlink{cmn_class_register_macros_8h_a7fbab4cbcdfbf35062de4f734e697034}{C\+M\+N\+\_\+\+D\+E\+C\+L\+A\+R\+E\+\_\+\+S\+E\+R\+V\+I\+C\+E\+S} and \hyperlink{cmn_class_register_macros_8h_a3dcd7732deee3f450741f0eb099ff778}{C\+M\+N\+\_\+\+D\+E\+C\+L\+A\+R\+E\+\_\+\+S\+E\+R\+V\+I\+C\+E\+S\+\_\+\+E\+X\+P\+O\+R\+T} are used in the header file to add the declaration of methods and data members within the scope of the registered class.
\item \hyperlink{cmn_class_register_macros_8h_a198d81818e0b66ebe53e692dc7c5d058}{C\+M\+N\+\_\+\+D\+E\+C\+L\+A\+R\+E\+\_\+\+S\+E\+R\+V\+I\+C\+E\+S\+\_\+\+I\+N\+S\+T\+A\+N\+T\+I\+A\+T\+I\+O\+N} and \hyperlink{cmn_class_register_macros_8h_aa48d094056bcdcf9e6956cb155d49df3}{C\+M\+N\+\_\+\+D\+E\+C\+L\+A\+R\+E\+\_\+\+S\+E\+R\+V\+I\+C\+E\+S\+\_\+\+I\+N\+S\+T\+A\+N\+T\+I\+A\+T\+I\+O\+N\+\_\+\+E\+X\+P\+O\+R\+T} are used in the header file to declare a template specialization of the global function cmn\+Class\+Services\+Instantiate for the registered class.
\item \hyperlink{cmn_class_register_macros_8h_a59997edde1868f5ce515ff85dfeaf2f8}{C\+M\+N\+\_\+\+I\+M\+P\+L\+E\+M\+E\+N\+T\+\_\+\+S\+E\+R\+V\+I\+C\+E\+S} and \hyperlink{cmn_class_register_macros_8h_a6d3e8cb412f66ff7e73f3ab52290f471}{C\+M\+N\+\_\+\+I\+M\+P\+L\+E\+M\+E\+N\+T\+\_\+\+S\+E\+R\+V\+I\+C\+E\+S\+\_\+\+T\+E\+M\+P\+L\+A\+T\+E\+D} are used in the source file to implement the methods declared by \hyperlink{cmn_class_register_macros_8h_a7fbab4cbcdfbf35062de4f734e697034}{C\+M\+N\+\_\+\+D\+E\+C\+L\+A\+R\+E\+\_\+\+S\+E\+R\+V\+I\+C\+E\+S} and the function specialization declared by \hyperlink{cmn_class_register_macros_8h_a198d81818e0b66ebe53e692dc7c5d058}{C\+M\+N\+\_\+\+D\+E\+C\+L\+A\+R\+E\+\_\+\+S\+E\+R\+V\+I\+C\+E\+S\+\_\+\+I\+N\+S\+T\+A\+N\+T\+I\+A\+T\+I\+O\+N}.
\end{DoxyItemize}

The macros ending with \+\_\+\+E\+X\+P\+O\+R\+T are required to handle the generation of D\+L\+Ls correctly with the different Microsoft compilers (see also \hyperlink{cmn_export_macros_8h_a99393e0c3ac434b2605235bbe20684f8}{C\+I\+S\+S\+T\+\_\+\+E\+X\+P\+O\+R\+T}) and the macro ending with \+\_\+\+T\+E\+M\+P\+L\+A\+T\+E\+D is required to register a specialized version of a templated class.

Finally, the definitions \hyperlink{cmn_class_register_macros_8h_ae4935a070cfc17d287f22f52d5141b17}{C\+M\+N\+\_\+\+D\+Y\+N\+A\+M\+I\+C\+\_\+\+C\+R\+E\+A\+T\+I\+O\+N} and \hyperlink{cmn_class_register_macros_8h_a655b277c7507ae5ce052a5134dacc596}{C\+M\+N\+\_\+\+N\+O\+\_\+\+D\+Y\+N\+A\+M\+I\+C\+\_\+\+C\+R\+E\+A\+T\+I\+O\+N} are provided to increase the code readability.
