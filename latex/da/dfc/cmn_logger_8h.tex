This macro is used to log human readable information within the scope of a global function (e.\-g. main()). It can also be used in classes which are not registered in \hyperlink{classcmn_class_register}{cmn\-Class\-Register} (see also macros \hyperlink{cmn_class_register_macros_8h_a7fbab4cbcdfbf35062de4f734e697034}{C\-M\-N\-\_\-\-D\-E\-C\-L\-A\-R\-E\-\_\-\-S\-E\-R\-V\-I\-C\-E\-S} and \hyperlink{cmn_class_register_macros_8h_a59997edde1868f5ce515ff85dfeaf2f8}{C\-M\-N\-\_\-\-I\-M\-P\-L\-E\-M\-E\-N\-T\-\_\-\-S\-E\-R\-V\-I\-C\-E\-S} declared in \hyperlink{cmn_class_register_8h}{cmn\-Class\-Register.\-h}). For a registered class, please use \hyperlink{cmn_logger_8h_a3fa9ee12c1a4d138c2594449aaf23729}{C\-M\-N\-\_\-\-L\-O\-G\-\_\-\-C\-L\-A\-S\-S}.

The message is streamed along with a Level of Detail to \hyperlink{classcmn_logger}{cmn\-Logger} provided that the function and overall log masks allow the level of detail associated to the message. The overall mask can be set using \hyperlink{classcmn_logger_aa1b05cddbe0f496c381453656a21af7d}{cmn\-Logger\-::\-Set\-Mask}. The function mask can be set using \hyperlink{classcmn_logger_afa246173cc26b1fa0bac407842dfd31c}{cmn\-Logger\-::\-Set\-Mask\-Function}.

The macro creates an output stream if the level of detail of the message satisfies the criterion defined above. It can be used as any other output stream\-:


\begin{DoxyCode}
\hyperlink{cmn_logger_8h_ad6bf31d204ebdfa9d018523c32ec5448}{CMN\_LOG\_INIT\_ERROR} << \textcolor{stringliteral}{"This is a message of LoD CMN\_LOG\_LEVEL\_INIT\_ERROR"} << argc << 
      std::endl;
\end{DoxyCode}

This macro is used to log human readable information within a registered class. It can only be used within the scope of a method part of a class registered in the \hyperlink{classcmn_class_register}{cmn\-Class\-Register} using the macros \hyperlink{cmn_class_register_macros_8h_a7fbab4cbcdfbf35062de4f734e697034}{C\-M\-N\-\_\-\-D\-E\-C\-L\-A\-R\-E\-\_\-\-S\-E\-R\-V\-I\-C\-E\-S} and \hyperlink{cmn_class_register_macros_8h_a59997edde1868f5ce515ff85dfeaf2f8}{C\-M\-N\-\_\-\-I\-M\-P\-L\-E\-M\-E\-N\-T\-\_\-\-S\-E\-R\-V\-I\-C\-E\-S} declared in \hyperlink{cmn_class_register_8h}{cmn\-Class\-Register.\-h}. For a non registered class of a global function, please use \hyperlink{cmn_logger_8h_a2eb3b40558ec17c9ae45197777aca7b2}{C\-M\-N\-\_\-\-L\-O\-G}.

The message is streamed along with a Level of Detail to \hyperlink{classcmn_logger}{cmn\-Logger} provided that two conditions are met\-:


\begin{DoxyItemize}
\item The overall log mask must allow the level of detail associated to the message. The overall mask can be set using \hyperlink{classcmn_logger_aa1b05cddbe0f496c381453656a21af7d}{cmn\-Logger\-::\-Set\-Mask}.
\item The class specific log mask must allow the level of detail associated to the message. Class specific masks can be set using \hyperlink{classcmn_logger_ac02d8a05e26687bf66951aab021adad7}{cmn\-Logger\-::\-Set\-Mask\-Class}, \hyperlink{classcmn_logger_ae835d053611487f98a3d216ca8ca6f28}{cmn\-Logger\-::\-Set\-Mask\-Class\-Matching} or \hyperlink{classcmn_logger_a88413527ca33537b0d13d1a9645c46a4}{cmn\-Logger\-::\-Set\-Mask\-Class\-All}.
\end{DoxyItemize}

The macro creates an output stream if the level of detail of the message satisfies the two criteria defined above. It can be used as any other output stream\-:


\begin{DoxyCode}
\hyperlink{cmn_logger_8h_a0c2eb3b8921d5e923d2b315d37b4f0b8}{CMN\_LOG\_CLASS\_INIT\_DEBUG} << \textcolor{stringliteral}{"This is a message of LoD 4"} << this->x << std::endl;
\end{DoxyCode}
