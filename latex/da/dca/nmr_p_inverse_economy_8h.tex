This function checks for valid input and calls the L\-A\-P\-A\-C\-K function. The approach behind this defintion of the function is that the user creates a data object from a code wherein it is safe to do memory allocation. This data object is then passed on to this method along with the matrix whose P\-Inverse is to be computed. The data object has members S, U, Vt and P\-Inverse, which can be accessed through calls to method $\ast$() along with adequate workspace for L\-A\-P\-A\-C\-K. This function modifies the contents of matrix A. For details about nature of the data matrices see text above.
