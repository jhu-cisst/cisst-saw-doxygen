\subsection{Detailed Description}
This class is not intended to be a top-\/level A\-P\-I. It has been provided to avoid making the templated S\-V\-D function as a friend of this class, which turns out to be not so easy in .N\-E\-T. Instead the \hyperlink{classnmr_s_v_d_dynamic_data_1_1_friend}{Friend} class provides a cumbersome way to get non-\/const references to the private data.
