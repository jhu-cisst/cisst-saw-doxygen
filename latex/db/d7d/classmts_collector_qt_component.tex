\subsection{Detailed Description}
Class used to translate cisst\+Multi\+Task commands and events to Qt slots and signals for components derived from \hyperlink{classmts_collector_base}{mts\+Collector\+Base} (i.\+e. \hyperlink{classmts_collector_state}{mts\+Collector\+State} and \hyperlink{classmts_collector_event}{mts\+Collector\+Event}). One can then use the Qt Widget \hyperlink{classmts_collector_qt_widget}{mts\+Collector\+Qt\+Widget} to control one or more data collection components. It is also possible to use a custom widget as long as it implements some or all slots and signals found in \hyperlink{classmts_collector_qt_widget}{mts\+Collector\+Qt\+Widget}.
