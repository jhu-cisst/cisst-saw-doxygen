Sends a serial break for a given duration in seconds.

On Linux, if the break duration is set to 0, the actual duration will be at least 0.\+25 seconds and at most 0.\+5 seconds.

On Mac O\+S, the implementation doesn\textquotesingle{}t use the break\+Length\+In\+Seconds. To provide a similar runtime, osa\+Sleep is used internaly. As the default duration on Mac\+O\+S it 0.\+4 seconds, the actual sleep time is break\+Length\+In\+Seconds -\/ 0.\+4.

On Windows, this method is implemented using Set\+Comm\+Break, osa\+Sleep and Clear\+Comm\+Break.
