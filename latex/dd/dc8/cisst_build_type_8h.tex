C\-I\-S\-S\-T\-\_\-\-B\-U\-I\-L\-D\-\_\-\-S\-U\-F\-F\-I\-X can be used to find a binary once compiled (where!). C\-Make creates sub-\/directories using the build type strings (e.\-g. Debug, Release, Rel\-With\-Deb\-Info, ...) for most I\-D\-E (Visual Studio, Xcode, ...). To find an executable, one can use \hyperlink{classcmn_path}{cmn\-Path} to look in C\-I\-S\-S\-T\-\_\-\-R\-O\-O\-T/bin/\-C\-I\-S\-S\-T\-\_\-\-B\-U\-I\-L\-D\-\_\-\-S\-U\-F\-F\-I\-X.

For other generators, changing the build type doesn't change the build destination (e.\-g. for G\-N\-U Make, executables go in C\-I\-S\-S\-T\-\_\-\-R\-O\-O\-T/bin).
