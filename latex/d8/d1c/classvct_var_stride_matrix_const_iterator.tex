Subtraction between iterators returns the number of increments needed for the second operand to reach the first operand, if it is reachable.

The number of increments is found by the following equations\+: Data\+Ptr -\/ (difference / Num\+Columns) $\ast$ Row\+Stride + (difference \% Num\+Columns) $\ast$ Column\+Stride == other.\+Data\+Ptr Data\+Ptr -\/ other.\+Data\+Ptr == (difference / Num\+Columns) $\ast$ Row\+Stride + (difference \% Num\+Columns) $\ast$ Column\+Stride

if (Row\+Stride $>$= Num\+Columns $\ast$ Column\+Stride) \{ (Data\+Ptr -\/ other.\+Data\+Ptr) / Row\+Stride == row\+\_\+diff == (difference / Num\+Columns) Data\+Ptr -\/ other.\+Data\+Ptr -\/ row\+\_\+diff $\ast$ Row\+Stride == (difference \% Num\+Columns) $\ast$ Column\+Stride (Data\+Ptr -\/ other.\+Data\+Ptr -\/ row\+\_\+diff $\ast$ Row\+Stride) / Column\+Stride == col\+\_\+diff == (difference \% Num\+Columns) difference == row\+\_\+diff $\ast$ Num\+Columns + col\+\_\+diff \} otherwise switch the roles of rows and columns.

\begin{DoxyNote}{Note}
this operation assumes reachability and does not test for it. 
\end{DoxyNote}
