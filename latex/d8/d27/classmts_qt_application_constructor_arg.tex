\subsection{Detailed Description}
\hyperlink{classmts_qt_application}{mts\+Qt\+Application} provides a component wrapper for the single Q\+Application object that must be created when using the Qt toolkit. This component is derived from \hyperlink{classmts_task_continuous}{mts\+Task\+Continuous}, and is set up to use the current (main) thread. The Q\+Application object is dynamically created in the class constructor, and the Run method calls Q\+Application\+::\+Exec, which does not return.
