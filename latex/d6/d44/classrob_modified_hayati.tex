\subsection{Detailed Description}
Hayati parameters of a coordinate frame. 

This class implements the Hayati kinematic model which is suitable for modelling both nearly parallel and nearly orthogonal axes. Compared to \hyperlink{classrob_hayati}{rob\-Hayati}, the convention of this model follows \mbox{[}siciliano2008springer\mbox{]} (Handbook of Robotics, Springer, 2008)
