This function creates and activates a \hyperlink{classcmn_get_char_environment}{cmn\-Get\-Char\-Environment} and deletes it at each call. If you need to get multiple characters you should probably use the \hyperlink{classcmn_get_char_environment}{cmn\-Get\-Char\-Environment} class.
